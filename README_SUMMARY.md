## Ringkasan Proyek

- **Masalah:** Ketimpangan sosial ekonomi antar kabupaten/kota di Sulawesi Tengah.
- **Pendekatan:** Segmentasi wilayah menggunakan indikator sosial ekonomi berbasis clustering.
- **Insight Utama:** Wilayah membentuk klaster pembangunan yang berbeda dan tidak acak.
- **Dampak:** Menjadi dasar penentuan wilayah prioritas dan kebijakan pembangunan terarah.
